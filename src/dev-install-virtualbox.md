# Virtualbox

VirtualBox是功能强大的x86和AMD64 / Intel64虚拟化产品，适用于企业和家庭使用。VirtualBox不仅是面向企业客户的功能极为丰富的高性能产品，而且还是根据GNU通用公共许可证（GPL）版本2的条款可作为开源软件免费提供的唯一专业解决方案。
